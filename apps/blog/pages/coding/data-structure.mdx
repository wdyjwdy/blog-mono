# Data Structure
## Stack
```js
class Stack {
    constructor() {
        this.items = []
    }
    
    push(item) {
        this.items.push(item)
    }
    
    pop() {
        return this.items.pop()
    }

    peek() {
        return this.items.at(-1)
    }
    
    size() {
        return this.items.length
    }

    isEmpty() {
        return this.items.length === 0
    }
}
```
## Queue
```js
class Queue {
    constructor() {
        this.items = []
    }

    enqueue(item) {
        this.items.unshift(item)
    }

    dequeue() {
        return this.items.pop()
    }
    
    front() {
        this.items.at(-1)
    }
    
    size() {
        this.items.length
    }

    isEmpty() {
        this.items.length === 0
    }
}
```
