# Function
## call()
- **描述**：修改函数的 this，并调用
- **语法**：`call(thisArg, ...args){:js}`
```js
function sum(a, b) {
    return this.initValue + a + b
}
sum.call({ initValue: 100 }, 1, 2) // 103
```
## apply()
- **描述**：修改函数的 this，并调用
- **语法**：`apply(thisArg, args){:js}`
```js
function sum(a, b) {
    return this.initValue + a + b
}
sum.apply({ initValue: 100 }, [1, 2]) // 103
```
## bind()
- **描述**：修改函数的 this，并返回该函数
- **语法**：`bind(thisArg, ...args){:js}`
```js
function sum(a, b) {
    return this.initValue + a + b
}
let bindSum = sum.bind({ initValue: 100 })
bindSum(1, 2) // 103
```
