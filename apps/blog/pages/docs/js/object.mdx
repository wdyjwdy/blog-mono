# Object
## Creating an Object
### with literals
- **描述**：创建一个新对象，其原型为 `Object.prototype`
- **语法**：`{}{:js}`
```js
let empty = {} // 空对象
let point = { x: 0, y: 0 } // 变量名作为键
let fruit = { 'name': 'apple', 'price': 10 } // 字符串作为键
```
### with new
- **描述**：创建一个新对象，其原型为 `fn.prototype`
- **语法**：`new fn(){:js}`
```js
let empty = new Object() // 等价于 {}
let empty = new Array()  // 等价于 []
```
### create()
- **描述**：创建一个新对象，其原型为 `proto`
- **语法**：`Object.create(proto, props?){:js}`
```js
Object.create({ a: 1 }) // 原型为 { a: 1 }
Object.create(Object.prototype) // 等价于 {}
Object.create(null) // 无原型对象
```
## Accessing an Object
### with dot
- **描述**：访问对象属性
- **语法**：`obj.prop{:js}`
```js
point.x
```
### with square bracket
- **描述**：访问对象属性
- **语法**：`obj.['prop']{:js}`
```js
point['x']
```
### query prototype chain
- **描述**：访问对象属性
```js
let obj = Object.create({ a: 1 })
obj.a // 1
```
## Modifying an Object
### with dot
- **描述**：修改对象属性
- **语法**：`obj.prop{:js}`
```js
point.x = 1
```
### with square bracket
- **描述**：修改对象属性
- **语法**：`obj.['prop']{:js}`
```js
point['x'] = 1
```
### with delete
- **描述**：删除对象的[自有属性]()
- **语法**：`delete obj.x{:js}`
```js
let obj = { a: 1 }
delete obj.a // true
```
## Checking an Object
### hasOwn()
- **描述**：检查对象的*自有属性*是否存在
- **语法**：`Object.hasOwn(obj, prop){:js}`
```js
let obj = Object.create({ a: 1 })
obj.b = 2
Object.hasOwn(obj, 'a') // false, 继承属性
Object.hasOwn(obj, 'b') // true, 自有属性
```
### with in
- **描述**：检查对象的*自有属性*和*继承属性*是否存在
- **语法**：`'prop' in obj{:js}`
```js
let obj = Object.create({ a: 1 })
obj.b = 2
'a' in obj // true, 继承属性
'b' in obj // true, 自有属性
```
## Enumerating an Object
### keys
- **描述**：返回一个数组，包含了对象[自有可枚举属性]()的键
- **语法**：`Object.keys(obj){:js}`
```js
Object.keys([1, 2, 3])      // ['0', '1', '2']
Object.keys({ a: 1, b: 2 }) // ['a', 'b'] 
```
### values
- **描述**：返回一个数组，包含了对象[自有可枚举属性]()的值
- **语法**：`Object.values(obj){:js}`
```js
Object.values([1, 2, 3])      // [1, 2, 3]
Object.values({ a: 1, b: 2 }) // [1, 2] 
```
### entries
- **描述**：返回一个数组，包含了对象[自有可枚举属性]()的键值对
- **语法**：`Object.entries(obj){:js}`
```js
Object.entries([1, 2, 3])      // [['0', 1], ['1', 2], ['2', 3]]
Object.entries({ a: 1, b: 2 }) // [['a', 1], ['b', 2]] 
```





### assign
- **描述**：将源对象的[自有可枚举属性]()复制到目标对象，并返回修改后的目标对象
- **语法**：`Object.assign(target, ...sources){:js}`
- **参数**
    - `target{:js}`：目标对象
    - `source{:js}`：源对象
```js
Object.assign({ a: 1 }, { b: 2 }) // { a: 1, b: 2 }
```
### fromEntries
- **描述**：将键值对转化为对象
- **语法**：`Object.fromEntries(iterable){:js}`
```js
Object.fromEntries([['a', 1], ['b', 2]]) // { a: 1, b: 2 }
```
### getPrototypeOf
- **描述**：返回对象的原型
- **语法**：`Object.getPrototypeOf(obj){:js}`
```js
let obj = Object.create({ a: 1 })
Object.getPrototypeOf(obj) // { a: 1 }
```
### setPrototypeOf
- **描述**：修改对象的原型
- **语法**：`Object.setPrototypeOf(obj, proto){:js}`
```js
let obj = Object.create({ a: 1 })
Object.setPrototypeOf(obj, { b: 2 })
Object.getPrototypeOf(obj) // { b: 2 }
```
### constructor
- **描述**：指向构造函数
- **语法**：`obj.constructor{:js}`
```js
function Animal() {}
let obj = new Animal()
obj.constructor // Animal
```
