# String
## Instance Methods
### at
- **描述**：返回索引位置字符串
- **语法**：`at(index){:js}`
- **参数**
    - `index = 0{:js}`：索引
        ```js filename="索引边界处理"
        if (index < 0) index = index + str.length
        ```
- **返回值**：`string`, `undefined`
```js
'hello'.at()   // 'h'
'hello'.at(0)  // 'h'
'hello'.at(-1) // 'o'

'hello'.at(10) // undefined
```

### slice
<Picture src="/img/docs/js/string/slice.png" />
- **描述**：提取子字符串
- **语法**：`slice(start, end?){:js}`
- **参数**
    - `start{:js}`：起点
    - `end{:js}`：终点
        ```js filename="索引边界处理"
        if (typeof start !== 'number') start = 0
        if (typeof end !== 'number') end = str.length
        if (index < 0) index = index + str.length
        if (index > str.length) index = str.length
        if (start > end) start = end
        ```
- **返回值**：`string`
```js
// 正向切片
'code-anchor'.slice() // 'code-anchor'
'code-anchor'.slice(5) // 'anchor'
'code-anchor'.slice(0, 4) // 'code'

// 逆向切片
'code-anchor'.slice(-1) // 'r'
'code-anchor'.slice(0, -1) // 'code-ancho'
```
### substring
<Picture src="/img/docs/js/string/slice.png" />
- **描述**：提取子字符串
- **语法**：`slice(start, end?){:js}`
- **参数**
    - `start{:js}`：起点
    - `end{:js}`：终点
        ```js filename="索引边界处理"
        if (typeof start !== 'number') start = 0
        if (typeof end !== 'number') end = str.length
        if (index < 0) index = 0
        if (index > str.length) index = str.length
        if (start > end) [start, end] = [end, start]
        ```
- **返回值**：`string`
```js
// 正向切片
'code-anchor'.substring() // 'code-anchor'
'code-anchor'.substring(5) // 'anchor'
'code-anchor'.substring(0, 4) // 'code'

// 逆向切片
'code-anchor'.substring('code-anchor'.length - 1) // 'r'
'code-anchor'.substring(0, 'code-anchor'.length - 1) // 'code-ancho'
```

### padStart
- **描述**：从左填充字符串
- **语法**：`PadStart(length, string?){:js}`
- **参数**
    - `length{:js}`：填充后，字符串的长度
    - `string = ' '{:js}`：填充的字符串
- **返回值**：`string`
```js

'hello'.padStart(10, '.')  // '.....hello'
'hello'.padStart(10, '._') // '._._.hello'
```
